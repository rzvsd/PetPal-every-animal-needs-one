# PetPal OSS References

This repo tracks OSS candidates and license decisions for PetPal.

## Approved Building Blocks

| Project | Use | License posture |
| --- | --- | --- |
| Expo | Mobile app runtime and tooling | Permissive, acceptable |
| Supabase | Auth, Postgres, storage, realtime, Edge Functions | Permissive/open-core, acceptable |
| React Native Paper | Mobile UI components | Permissive, acceptable |
| Zod | Shared validation | Permissive, acceptable |
| TanStack Query | Data fetching/cache | Permissive, acceptable |
| React Hook Form | Forms | Permissive, acceptable |

## Reference Only

Do not copy code from full dating/adoption clones unless a license review explicitly approves it. GPL, AGPL, no-license, and unclear template licenses are reference-only by default.

## Product Rule

PetPal should reuse OSS infrastructure and libraries, not rebrand an existing dating clone.
