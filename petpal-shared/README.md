# PetPal Shared

Shared TypeScript contracts for PetPal apps and backend functions.

This package is the source of truth for:

- Product modes and feature flags
- User roles and organization verification inputs
- Animal and Adopt/Foster listing creation inputs
- Discovery filters, public listing shape, and application review inputs
- Listing and verification statuses
- Dog/cat-only v1 rules
- Public/private data boundary helpers
- Zod validation schemas used by clients and backend
